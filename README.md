## 第二周
### 物件规则：
- [x] 1. 射击命中方块，获得积分X (score) 分
- [x] 2. 子弹命中方块后销毁
- [x] 3. 方块被子弹命中后，缩放为Y (hitScale) 倍，再次被命中后销毁

### 游戏流程：
- [x] 1. 游戏开始时随机N (importantGoalCount) 个方块成为“重要目标”，射击命中后获得双倍积分
- [x] 2. 游戏开始后限时T (Seconds) 秒，时间到后游戏结算，打印日志输出每个玩家获得的积分和所有玩家获得的总积分
